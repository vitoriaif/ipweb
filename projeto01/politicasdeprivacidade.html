Política de Privacidade

A JM Viagens está comprometida com a proteção da privacidade e segurança dos dados pessoais dos seus clientes. Esta Política de Privacidade explica como coletamos, usamos, compartilhamos e protegemos seus dados pessoais.

Dados Pessoais Coletados

Coletamos dados pessoais quando você utiliza nossos serviços, como:

  • Nome completo
  • Endereço de e-mail
  • Número de telefone
  • Endereço residencial
  • Data de nascimento
  • Dados de pagamento
  • Informações sobre suas viagens

Como Usamos Seus Dados Pessoais

Usamos seus dados pessoais para:

  • Fornecer os serviços que você solicitou
  • Enviar informações sobre ofertas e promoções
  • Melhorar nossos serviços
  • Realizar pesquisas de mercado
  • Atender a solicitações legais

Compartilhamento de Dados Pessoais

Compartilhamos seus dados pessoais com:

  • Empresas parceiras para fornecer os serviços que você solicitou
  • Autoridades legais quando necessário

Segurança dos Dados Pessoais

A JM Viagens utiliza medidas de segurança para proteger seus dados pessoais contra acesso não autorizado, uso indevido, divulgação, alteração ou destruição.

Seus Direitos

Você tem o direito de:

  • Acessar seus dados pessoais
  • Corrigir seus dados pessoais
  • Solicitar a exclusão de seus dados pessoais
  • Limitar o uso de seus dados pessoais
  • Opor-se ao uso de seus dados pessoais
  • Solicitar a portabilidade de seus dados
